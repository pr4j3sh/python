# Python Package/Library

A `python` package/library template

## Installation

```bash
npm create @pr4j3sh/frames@latest python mypkg
```

```bash
python -m venv .venv
```

```bash
source .venv/bin/activate
```

```bash
pip install -r requirements.txt
```

## Usage

```bash
python -m package.main
```
